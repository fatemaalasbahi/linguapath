# LinguaPath Feature Specification


# MVP Features


## 1. Authentication

Users can:

- create an account
- login
- logout

Each user has their own private learning profile.


---

# 2. Exam Goal Setup

Users enter:

- exam type
- current score/level
- target score
- exam date


Example:

Exam:
Duolingo English Test

Current score:
95

Target:
120

Deadline:
3 months


---

# 3. AI Diagnostic Assessment

LinguaPath provides an initial assessment.

Example:

Writing prompt:

"Describe an important challenge you experienced."


AI analyzes:

- grammar
- vocabulary
- organization
- estimated proficiency score


Example output:

Estimated Score:
105


Strengths:

- clear structure


Weakness:

- limited vocabulary


Recommendations:

- practice academic vocabulary


---

# 4. Personalized Study Plan Generator

The AI creates a roadmap based on:

- current level
- target score
- deadline
- available study time


Example:

Week 1:

Writing:
- improve paragraph structure

Vocabulary:
- learn academic vocabulary

Speaking:
- practice responses


---

# 5. AI Practice Feedback

Students submit:

- writing responses
- practice answers


AI provides:

- evaluation
- corrections
- improvement suggestions


---

# 6. Progress Dashboard

Dashboard displays:

- current estimated score
- target score
- progress percentage
- completed tasks
- weak skills


---

# 7. Notifications

The system sends:

- study reminders
- progress updates


Example:

"You have a writing practice task today."


---

# 8. Feedback System

Users can submit:

- suggestions
- bugs
- comments


The feedback:

- saves to database
- sends notification to Slack


---

# 9. Landing Page

The website includes:

- introduction
- benefits
- how it works
- call-to-action
